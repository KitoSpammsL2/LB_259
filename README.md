# Vorhersage von Herzkrankheiten mit Machine Learning

## Beschreibung des Datensatzes

Dieser Datensatz enthält strukturierte klinische und diagnostische Patientendaten zur Vorhersage von Herzkrankheiten. Die enthaltenen Merkmale umfassen unter anderem Alter, Geschlecht, Art der Brustschmerzen, Blutdruck, Cholesterinwert, Nüchternblutzucker, EKG-Ergebnisse, maximale Herzfrequenz, belastungsinduzierte Angina sowie ST-Depression. Ziel des Projekts ist es, anhand dieser medizinischen Messwerte vorherzusagen, ob eine Person an einer Herzkrankheit leidet. Der Datensatz enthält über 270’000 Einträge und eignet sich für eine überwachte Klassifikationsaufgabe.

## Datenschutz

Der Datensatz enthält keine personenbezogenen Informationen wie Namen, Adressen oder Identifikationsnummern. Es handelt sich ausschliesslich um anonymisierte medizinische Messwerte. Dadurch besteht kein direkter Personenbezug und kein relevantes Datenschutzrisiko.
